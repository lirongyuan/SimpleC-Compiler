Simple C Compiler
===================================
Project link: http://www.cs.purdue.edu/homes/grr/cs250/lab7-comp/

A compiler for the Simple C language.
It could generate x86-64 assembly language that can be assembled to produce an executable file.

Setup:
------
```
make
```

Run the tests:
-----------------
```
make tests
```

```
cd tests
testall
```
